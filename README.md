# SIDISI SMKN 6

**SISTEM DATA INFORMASI SISWA SMKN 6**  
Sebuah website untuk mempermudah manajemen data siswa, jurusan, agama, dan pengguna di SMKN 6.

- **Data Jurusan**: Kelola data jurusan di SMKN 6.
